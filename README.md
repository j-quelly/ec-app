# ec-app
- A single-page client-side application that allows a teacher to browse homework assignments and submissions.

### Current Version: 1.1.1

## Getti
1. ```$ npm install```
2. ```$ npm start```
3. Open browser to http://localhost:3000

## Default Usage


## Alternative Usage
```$ grunt serve-dev```
